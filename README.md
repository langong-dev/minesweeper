# LanGong Minesweeper

> The minesweeper game for Linux and MacOS.


# Install Game

```bash
$ wget https://cdn.jsdelivr.net/gh/langong-dev/minesweeper/install.sh
# bash install.sh
```

# Creat user

each user in your computer can creat only **one** user.

```bash
$ MineSweeper init
```

# Start game

**Don't** use **root** to run the game.

```bash
$ MineSweeper
```
